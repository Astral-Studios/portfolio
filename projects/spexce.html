Spexce Website

Visit Website

Project Overview

The Spexce Website serves as the central hub for the Spexce ATM10 Minecraft Server community. Built on the NamelessMC platform, this website provides players with access to server information, news updates, staff applications, and community features.

Key Features

  • Dynamic news post system for server announcements and updates using the Ghost Module.
  • Comprehensive staff application forms with custom fields.
  • Server status monitoring and real-time player counts.
  • Integrated Discord connection for unified community experience.
  • Custom theme - Cobalt Theme from https://coldfiredzn.com/

Technologies Used

  • NamelessMC - Content management system
  • MySQL - Database management
  • HTML/CSS - Frontend customization

Community Integration

Successfully integrated Discord widgets and server status displays, creating a seamless connection between the website and other community platforms. Implemented automated news posting systems and configured application workflows that streamline staff recruitment processes. The website serves as the primary information source for new and existing players.
