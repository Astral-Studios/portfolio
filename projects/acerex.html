Project Overview

As Chief Operating Officer at AceRex Network, I managed the complete technical infrastructure for a growing Minecraft server community. This role required balancing multiple systems and ensuring optimal performance across all platforms.

Key Responsibilities

  • Managing Pterodactyl Panel for server orchestration and resource allocation.
  • Overseeing Discord server with custom bots and moderation systems.
  • Maintaining website infrastructure and ensuring 99.9% uptime.
  • Managing store infrastructure and payment processing.
  • Optimizing server performance - RAM, CPU Threads, Storage Allocation.
  • Implementing backup systems and disaster recovery protocols.

Technologies Used

  • Pterodactyl Panel - Server management and deployment
  • Linux (Debian 12) - Server Operating Systems
  • Nginx - Web Server and Reverse Proxy
  • MySQL / MariaDB - Database Management
  • Docker - Containerization and Deployment

Achievements

Successfully managed infrastructure supporting over 50 concurrent players, reduced server downtime by 95%, and implemented automated backup systems that prevented data loss during critical incidents.
